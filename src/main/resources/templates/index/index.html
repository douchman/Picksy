Picksy 가 무엇인가요?

Pick + -sy: 선택을 더 친근하게, 재미있게!

Pick:

선택하다, 고르다

-sy:

Bitsy, Tipsy처럼 부드럽고 캐주얼한 느낌

Picksy는 사용자가 만든 대결주제에 대해
다양한 엔트리(항목)들을 1:1로 비교하며 선택해 나가는
취향 기반 토너먼트 플랫폼입니다.

Picksy 용어사전 📚

🧩 대결주제

사람들이 선택을 진행할 중심 주제에요.

예: "25년 최고의 간식은 ?", "가장 귀여운 고양이는?"
🎯 엔트리

각 주제를 구성하는 항목이에요. 경기의 후보들이죠.

예: "치킨", "떡볶이", "하품하는 고양이"
📊 통계

선택한 결과들이 집계되어 실시간 랭크와 데이터로 확인돼요.

왜 Picksy 일까요?

🎨 다양한 미디어 지원

🖼️이미지, 🎬 영상,  유튜브 링크까지 자유롭게 섞어
여러분만의 대결 항목을 만들 수 있어요!

⚛ AI와 함께, 더 스마트하게

앞으로는 주제에 딱 맞는 이미지도
AI가 자동으로 생성해 드릴 거예요.

🏆 세부 통계 제공

누가 더 많이 이겼을까요?
엔트리 간 상성, 선택률, 승률을 한눈에 확인!

Picksy는 이렇게 시작해요

1️⃣ Pick it

대결 엔트리중 중 여러분의 선택을 고르세요

2️⃣ Rank it

모두의 선택이 랭크로 집계돼요

3️⃣ Play it

다양한 주제로 즐기세요

지금 여러분의 취향을 선택해보세요
